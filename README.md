## 🌟 Особенности

- 🎥 **Просмотр в высоком качестве**: Поддержка современных видеоформатов.
- 🎨 **Современный UI/UX**: Интуитивно понятный, красивый и плавный интерфейс.
- 🌙 **Темная тема**: Идеально для просмотра фильмов в вечернее время.
- ⚡ **Высокая производительность**: Оптимизировано для быстрой работы на любых ПК.
- 🔍 **Удобный поиск**: Быстро находите нужные фильмы по названию или жанру.

## 🚀 Установка

Пока проект находится в стадии разработки, вы можете запустить его из исходного кода.

### Требования
- [Node.js](https://nodejs.org/) (версии 16+)
- Git

### Шаги

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/openfilms.git
   ```

2. **Перейдите в папку проекта:**
   ```bash
   cd openfilms
   ```

3. **Установите зависимости:**
   ```bash
   npm install
   ```

4. **Запустите приложение:**
   ```bash
   npm run dev
   ```

## 🛠 Разработка
